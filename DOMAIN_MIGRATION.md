## Future: redirect other owned domains → gameset-hk.com

Owner has additional domains they want to point at GameSet HK. Easiest path is **Cloudflare Bulk Redirects** (Free plan supports 20 redirects).

For each old domain:

1. Add the domain to Cloudflare (if not already there) and update its registrar nameservers to Cloudflare's
2. **Rules → Bulk Redirects → Create List** named e.g. `to-gameset-hk`
3. Add an entry per old domain:
   - Source URL: `olddomain.com/*`
   - Target URL: `https://gameset-hk.com/$1`
   - Status code: **301** (permanent)
   - Preserve query string: ✓
   - Subpath matching: ✓
4. **Rules → Bulk Redirects → Create Rule** → bind the list to all incoming traffic
5. (Optional) Add the same for `www.olddomain.com/*` → `https://gameset-hk.com/$1`
